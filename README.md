# Blox Fruits Script

Script hỗ trợ chơi Blox Fruits với nhiều tính năng hữu ích.

## Tính năng chính
- Tự động farm
- Tự động làm nhiệm vụ
- Hỗ trợ nhiệm vụ ẩn
- Dịch chuyển đến các đảo
- Công cụ dịch ngôn ngữ chat
